王道5.3节03. 编写后序遍历二叉树的非递归代码

示例 1：

```
输入：root = ["1","null","2","null","null","3"]
输出：[3,2,1]
```

示例 2：

```
输入：root = []
输出：[]
```

示例 3：

```
输入：root = ["1"]
输出：[1]
```

示例 4：

```
输入：root = ["1","2"]
输出：[2,1]
```

示例 5：

```
输入：root = ["1","null","2"]
输出：[2,1]
```

示例 6：

```
输入：root = ["1", "2", "3", "null", "4", "5", "6", "null", "null", "null", "null", "null", "null", "7"]
输出：[4,2,5,7,6,3,1]
```
